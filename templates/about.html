About MediScribe AI

Advanced Medical Prescription OCR System

Project Overview College Project

MediScribe AI is an advanced Optical Character Recognition (OCR) system specifically designed to extract and process information from handwritten medical prescriptions. The system utilizes multiple image preprocessing techniques and specialized OCR algorithms to achieve high accuracy rates, even with challenging doctor's handwriting.

This project addresses the real-world challenge of digitizing medical prescriptions, which can help reduce medication errors, streamline pharmacy workflows, and improve patient safety.

Key Features:
  • Advanced Image Preprocessing: Applies multiple filters and transformations to enhance handwriting visibility and improve OCR accuracy.
  • Medical Dictionary Integration: Uses a comprehensive database of medication names and terminology to improve recognition and correct errors.
  • Machine Learning Models: Employs CNN (Convolutional Neural Network) models for enhanced character recognition.
  • Training System: Allows users to train the system with specific images for 100% recognition accuracy on subsequent uploads.
  • User-Friendly Interface: Provides an intuitive web interface for uploading prescriptions and viewing results.

System Workflow
  1. Prescription image is uploaded through the web interface
  2. Image preprocessing enhances quality and readability
  3. OCR engine extracts raw text from the processed image
  4. Text preprocessing cleans and normalizes the extracted text
  5. Medical dictionary matching identifies medications and dosages
  6. Results are displayed in a structured, user-friendly format
